Creating Forms

An essential feature contained in most modern web sites is the ability for users to submit input to the website. This is done through the use of web forms. According to Jeffery Brown, Chris Minnick, and Susan Lane (2010), in the book Design Methodology and Technology (Adobe CS5 Web Edition), Dreamweaver makes creating input forms so easy one does not need much knowledge of X/HTML to accomplish this task (p. 22-5). This tutorial is going to discuss how to use Dreamweaver to create user input forms.

Now, your web site should be looking fairy decent. In the last tutorial you added content to your web pages now it is time to create a mechinism that allows the user to communicate back with you. User input forms are the technology that allow the user to this. DreamWeaver provides great tools to build web forms. First thing to do is to select the element that you want to create the form in. Click on the insert tab and select forms. There you will find various elements that can be inserted into a form. The purpose of your form will determine what elements to place into it. Dreaweaver does not provide server side form validation, but it does provide client side validation using JavaScript. Now you are able to build a form so your users can submit data through your website, congratulations!
