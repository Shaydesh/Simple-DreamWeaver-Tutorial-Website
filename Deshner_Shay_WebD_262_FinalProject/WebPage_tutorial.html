Create a Web Page

According to Jeffery Brown, Chris Minnick, and Susan Lane (2010), in the book Design Methodology and Technology (Adobe CS5 Web Edition), there are two different methods a developer can layout web pages with Adobe DreamWeaver. These methods are through tables and CSS positioning. However, CSS positioning is the prefered method (p.20-4). Brown, Minnick, and Lane go on to teach that in Dreamweaver webpage layout is made very easy through the use of absolutly positioned html elements called AP elements (p. 20-4). This tutorial will go over the steps to creating your first web page with Dreamweaver, and discovering the power of the AP element.

In Dreamweaver a developer can inport an existing web page and make alterations to it, or they can create a completly new web page. To create a completly new web page from scratch, in the DreamWeaver title menu, in the create new colunm, click on HTML. The developer can also go to the file tab, click new, and select blank page from the menus. If you plan on creating an entire website, click on the site tab, and select new site. Dreamweaver will generate the file system nessesary to accomplish this. In order to insert AP objects into to your newly formed webpage to create a layout. Simply, click on the Insert tap, select layout objects, and then AP Div. Filling your newly formed web page with content is now up to you!... Brown, J., Minnick, C., Lane, S. (2010). Design Methodology and Technology(Adobe CS5 Web Edition). Prosoft Learning.
