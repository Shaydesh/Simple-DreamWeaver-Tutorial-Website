Welcome to the Adobe Dreamweaver Tutorial

This tutorial is meant to help developers and those new to web development create a website using Adobe DreamWeaver. According to Jeffery Brown, Chris Minnick, and Susan Lane (2010), in the book Design Methodology and Technology (Adobe CS5 Web Edition), Dreamweaver is a graphical user interface web page editor and a web site management tool (p. 22-3). I hope that this tutorial can help begin the journey to creating your very own website.

This tutorial will start out by teaching you how to create a simple html page and layout the elements on to the newly created HTML page using the AP element tool. It will then go over how to create a website directory, a webpage template, and how to create new web pages based on the template you created. The tutorial will then discuss how to insert and format text, images, and how create hyperlinks and image maps out of those images. The final part of the tutorial will teach how to collect user input using webforms, and how to create webforms to accomplish this task. If you are ready to get started go ahead and click on the next page link to begin with web page creation!
