Inserting and Formatting Text, Images, and Hyperlinks

Dreamweaver's tools provide easy ways to format text, import images, create Image maps, and turn images into hyperlinks. This tutorial is going to go over the process of inserting text from a word document while maintaining it's original formatting, inserting images into a webpage, assigning hyperlinks to images, and creating an image map.

So, you have created your original web page, created a template out of it, and generated web pages based on the template in order to populate your website. Now, it is time to add content to your web pages. One of the tasks you may want to perform is importing text from a Microsoft word document. Dreamweaver provides excellent tools that allow the developer to accomplish this task while at the same time maintaining the formatting of the text that was defined in Microsoft Word. In order to do this, select the element you want to import the text into, click on the file tab, select import, and then choose word document. Navigate to the document you wish to import and select it. Dreamweaver imports the contents of the document and converts it's content to X/HTML compliant code. Inserting Images into your web page is just as easy. Click on the insert tab, select image and then navigate to the image you wish to insert into your webpage. Once the image is inserted, in the properties tab at the bottom of the Dreamweaver console, in the field named link, the developer can enter a URL that will turn the image into a hyper link for the entered URL. In order to create an image map, or select just a certain peice of an image to act as a hyperlink, the developer needs to choose one of the hot spot tools represented by an arrow, a rectangle, a circle, and a polygon,located in the lower left hand side of the properties tab. Now, highlight the area of the image that you want to create a hyperlink out of and define the url that will be associated with that link. You have now aquired the power to import text, images, and to create hyperlinks out of your images!
